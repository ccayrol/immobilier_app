# 🏘️ Immobilier App

Une application web développée pour un **rentier immobilier bordelais**, avec pour objectif de **renforcer sa présence en ligne** et de véhiculer une **image haut de gamme et professionnelle**.

🔗 **Accéder à l'application** : [immobilier-app.vercel.app](https://immobilier-app.vercel.app/)

---

## ✨ Présentation

Ce projet vise à valoriser un portefeuille immobilier à travers une interface moderne, élégante et intuitive.  
Il permet à l'utilisateur final (clients, prospects, partenaires) de consulter les biens en toute simplicité, tout en renforçant la visibilité en ligne du propriétaire.

---

## ⚙️ Stack technique

### Frontend
- **React**
- **Tailwind CSS**

### Backend
- **Node.js**
- **Firebase** (base de données)

---

## 🔧 Fonctionnalités principales

- Affichage des biens immobiliers avec visuels et descriptions détaillées
- Interface responsive (adaptée aux mobiles et tablettes)
- Architecture moderne avec composants réutilisables
- Connexion dynamique à la base de données Firebase


# Installer les dépendances
npm install

# Lancer le serveur de développement
npm start
